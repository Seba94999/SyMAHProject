# Skill: Transacciones Financieras

El sistema maneja cuatro tipos de transacciones:

Pago
Cobro
Gasto
Prestamo

Todas las transacciones deben:

- registrarse en el libro diario
- impactar cuentas contables
- tener fecha
- mantener historial

Las transacciones no se modifican.

Solo pueden darse de baja.
