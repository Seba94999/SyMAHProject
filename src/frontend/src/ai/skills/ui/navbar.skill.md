# Skill: Navbar

## Responsabilidad

Renderizar la barra de navegación superior fija.

## Estructura

- Posición: fija superior
- Siempre visible
- Contiene navegación entre páginas:
  - Inicio
  - Clientes
  - Trabajos
  - Empleados
  - Transacciones
  - Cuenta SyMAH

## Comportamiento

- Resaltar pestaña activa
- Permitir navegación entre páginas

## Estilo

- Fondo oscuro
- Item activo destacado (dorado)

## Props esperadas

- activePage
- onNavigate
