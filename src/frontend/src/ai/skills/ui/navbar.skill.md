# Skill: Navbar

## Responsabilidad

Renderizar la barra de navegación principal de SYMAH.

La navbar debe funcionar como el eje de navegación global del sistema y mantenerse siempre visible.

---

# Filosofía Visual

La navbar debe sentirse como:

- barra de control empresarial
- navegación premium
- sistema operativo de gestión
- panel de acceso principal

Evitar:

- menús básicos
- navegación desordenada
- exceso de elementos
- estilos planos

---

# Objetivo

La navbar debe permitir:

- navegación clara entre módulos
- acceso rápido
- orientación visual
- identificación de sección activa

---

# Estructura

La navbar debe incluir:

- logo o nombre del sistema
- navegación principal
- indicador de página activa
- acciones rápidas opcionales

---

# Posicionamiento

## Reglas

- posición fija superior
- siempre visible
- ancho completo
- prioridad visual alta

---

# Navegación

## Páginas Principales

La navbar debe incluir navegación hacia:

- Inicio
- Clientes
- Trabajos
- Empleados
- Transacciones
- Cuenta SyMAH

---

## Comportamiento

La navegación debe:

- permitir cambio de página
- mantener estado activo
- soportar routing dinámico

---

# Estado Activo

## Reglas

La página activa debe:

- destacarse visualmente
- usar color dorado
- incluir indicador visual
- mantener claridad de navegación

---

## Indicadores posibles

- underline
- glow
- background destacado
- badge lateral

---

# Estilo Visual

## Fondo

- oscuro premium
- translúcido opcional
- backdrop blur
- borde inferior sutil

---

## Items

Los items deben:

- mantener spacing uniforme
- usar tipografía clara
- incluir hover states
- soportar iconografía opcional

---

# Hover

Los elementos de navegación deben:

- mejorar contraste
- mostrar glow suave
- usar transición fluida

---

# Responsive

La navbar debe:

- adaptarse a distintos tamaños
- soportar menú mobile
- reorganizar elementos correctamente

---

# Props Esperadas

- activePage
- onNavigate
- navigationItems opcional

---

# Integración

La navbar debe integrarse correctamente con:

- pageLayout
- layout
- dashboard
- routing
- floatingButton

---

# UX

La navbar debe:

- facilitar orientación
- reducir fricción de navegación
- destacar sección actual
- mantener acceso rápido a módulos

---

# Reglas Visuales

La navbar debe usar:

- dark theme
- acentos dorados
- spacing consistente
- transiciones suaves
- jerarquía visual clara

---

# Restricciones

Nunca:

- usar estilos inline
- ocultar navegación principal
- generar menús desordenados
- romper el design system

---

# Objetivo Final

La navbar debe convertirse en el centro de navegación principal de SYMAH, manteniendo una experiencia enterprise moderna, clara y consistente.
